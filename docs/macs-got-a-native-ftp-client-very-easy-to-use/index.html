Mac's got a native FTP client - very easy to use

Feb 7, 2011

While finder is active (select the desktop), press ⌘+k (cmd+k) and type ftp://your_host.com.

You will be able to reach it via /Volumes/your_host.com. A bit easier than using sshfs.. Via osxdaily.com
