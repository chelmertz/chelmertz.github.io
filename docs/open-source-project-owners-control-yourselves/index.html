Aug 12, 2010

  • irc
  • email lists
  • twitter
  • forum
  • blog
  • facebook
  • github/jira
  • wiki

and not least, in official sources quickly multiplies the difficulty of focusing in the right spot. The same thing goes for official sources concerning a submodule/sister of the main project.
