Open file: cmd + down; navigate upwards: cmd + up

Jun 2, 2010

When navigating a finder window (a directory) you go up one level with cmd + up and open a file with cmd + down.

It gets trickier for each day, but pretty consistent behaviour if not logical at all times. At least it doesn’t clutter the UI.
