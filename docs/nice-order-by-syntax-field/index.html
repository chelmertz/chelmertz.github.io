Nice ORDER BY-syntax: FIELD()

Nov 29, 2010

SELECT turtle, pizza, rating FROM heroes ORDER BY FIELD(turtle, ‘Donatello’, ‘Michelangelo’, ‘Raphael’, ‘Leonardo’)

This let’s you get the ratings of Donatello’s first, etc. Really practical.
