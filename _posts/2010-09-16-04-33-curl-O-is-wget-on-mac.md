If you want to download a file:

    curl -O an-url

Exactly the same as `wget an-url` on Ubuntu.

Via [A wget replacement for MAC OS X - Snow Leopard Mac OS X](http://www.mymacosx.com/terminal/wget-replacement-macos.html)
