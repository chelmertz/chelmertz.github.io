<code>which</code> tell you where a program is located.
<div class="CodeRay">
<div class="code">
<pre>$ which php
/usr/bin/php</pre>
</div>
</div>
<em>Very</em> helpful if you install a lot of stuff which is put in “random” locations as /usr/bin, /bin, /usr/local/bin, /opt … the list goes on.
